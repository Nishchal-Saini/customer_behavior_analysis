# customer_behavior_project (end-to-end analysis)

## overview
This project analyzes customer shopping behavior using a full end-to-end data analytics workflow. It combines Python, SQL, and Power BI to transform raw data into meaningful business insights and visualizations.
The goal is to understand customer purchasing patterns, product performance, and revenue trends to support data-driven decision making.

## Dataset
- <a href = "https://github.com/Nishchal-Saini/customer_behavior_analysis/blob/main/customer_shopping_behavior.csv">Dataset<a>

The dataset contains customer-level shopping data including:
-Customer ID
-Age
-Gender
-Item Purchased
-Category
-Purchase Amount(USD)
-Location
-size
-color
-season
-review rating
-subscription status
-Shipping type
-discount applied
-promo code used
-previous purchases
-Payment Method
-Frequency of purchases
The raw data is loaded and cleaned using Python before being stored and queried in a SQL database.

## Tools & Technologies
-Python (Pandas, NumPy):	Data loading, EDA, and cleaning
-MySQL:	Data storage and SQL queries
-SQL:	Business queries and aggregations
-Power BI:	Dashboard and data visualization
-Gamma:	Presentation (PPT)
-Jupyter Notebook:	Analysis and documentation
